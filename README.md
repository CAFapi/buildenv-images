# openSUSE Build Environment Maven Images

This project builds a number of openSUSE-based images that include various versions of JDK, C++, Dotnet and OpenTofu along with Maven. They are intended to be used to build other images.

The images which are built in this project are:
- [buildenv-jdk17](buildenv-jdk17-image)
  - [buildenv-dotnet](buildenv-dotnet-image)
- [buildenv-jdk21](buildenv-jdk21-image)
  - [buildenv-cpp](buildenv-cpp-image)
  - [buildenv-tofu](buildenv-tofu-image)
- [buildenv-jdk25](buildenv-jdk25-image)
